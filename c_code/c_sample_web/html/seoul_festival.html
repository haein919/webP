제19회 서울디자인페스티벌
2020/12/9 - 2020/12/13

50 days 06 hours 11 minutes 48 seconds

서울디자인페스티벌은 1976년부터 축적되어 온 월간 <디자인>의 콘텐츠와 국내외 디자인분야 네트워크를 기반하여 2002년 처음 개막한 디자인 전문 전시입니다. ‘디자이너 프로모션’을 모토로 디자이너, 디자인 브랜드, 기업들과 함께 국내외 디자인의 동향을 선보이고, 국내 디자인 경쟁력을 강화시키기 위해 기획된 행사로 한국 디자이너들의 세계 진출을 돕고 세계 디자이너들을 국내에 소개하는 교류의 장으로 역할 해 왔습니다.
